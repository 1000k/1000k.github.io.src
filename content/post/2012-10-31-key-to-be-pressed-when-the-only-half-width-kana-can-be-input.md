「Alt + Tab」キーなどを押し損じて、いきなり半角カナ入力しかできなくなることがあるのでメモ。

  * **Ctrl + CapsLock**: ローマ字/カナ入力変更
  * **Alt + CapsLock**: ローマ字/カナ入力変更（トグル）

ちなみに、Shift + CapsLock はローマ字の大文字/小文字のトグルです。

必要ないからキーをもぎ取ってハンダで埋めてしまいたい。
